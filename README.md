# charliek.github.io

Project index page for Charlie Knudsen's GitHub projects.

## Development

Install dependencies:

```bash
uv sync --group docs
```

Serve locally:

```bash
uv run mkdocs serve
```

Build static site:

```bash
uv run mkdocs build
```

## Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the `main` branch.
